# 📝 메모정 (made by minjun"Jeong")

Spring boot와 MySQL을 활용한 미니 메모장 서비스


## 🚀 주요 기능 (Key Features)

- **메모 관리 (CRUD)**: 메모 작성, 목록 조회, 수정 및 삭제 기능
- **사용자 인증 (Authentication)**: Spring Security를 활용한 관리자 로그인 및 로그아웃
- **커스텀 디자인**: Bootstrap 5를 활용한 UI 디자인
- **데이터 영구 저장**: MySQL DB 연동을 통한 데이터 보관

## 🛠 사용 기술 (Tech Stack)

- **Backend**: Java 23, Spring Boot 4.0.2, Spring Data JPA, Spring Security
- **Frontend**: Thymeleaf, Bootstrap 5, HTML/CSS
- **Database**: MySQL
- **Build Tool**: Gradle

## 📸 실행 화면

- **메인 화면**: 저장된 메모 리스트 확인 및 새 메모 작성
- **로그인 페이지**: 맞춤형 디자인이 적용된 관리자 전용 로그인 창
